About

Eugene T. Mahoney State Park App is a project for GEOG777 that provides an interactive map and tools for exploring Mahoney Park in Nebraska. The app allows visitors to view trail maps, facilities, events, and community-contributed features for an enhanced park experience. located near Ashland between Omaha and Lincoln, is a premier, year-round Nebraska state park known for its extensive family-friendly activities. It features a, aquatic center, marina, indoor climbing wall, horseback riding, hiking, and extensive winter activities like sledding and ice skating

The app was built by David Vanosdall for GEOG777 in 2026, using Bootswatch Slate theme, Mapbox for mapping, and custom interactive tools.

  • Interactive Map powered by Mapbox GL JS
  • Toggleable feature layers for trails, facilities, natural features, and events
  • User-submitted community data layer
  • Mobile-friendly responsive design
